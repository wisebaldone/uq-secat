About

What?

This website is an information dump for the collection of Student Evaluations of Course and Teachers at the University of Queensland. The course information is made public to students here while teacher evaluations are kept private to the university. This website is not affiliated with the University of Queensland.

Why?

The viewer for the results that the university hosts is pretty janky and takes forever to get to the course you'd like to view. This website was made as a portfolio piece to show an alternate way of accessing the information.

Who?

My name is Evan Hughes, I'm a Engineer & Science undergrad at UQ studying Electrical & Computer Engineering and Computer Science. See www.wisebaldone.com for more of my work or more about me.

How?

The project gathers no data of its own and collects it all from the offical UQ page where it is skimmed using a python script found here. The scraper itself uses selenium due to the target website having an internal state machine for display information. This made it difficult to use some http posts to get the information. Instead the skimmer manually browses every page for every course. A json file is output for each semester the course has results and a pseudo static api is formed which is explained more on the api portion of the site.

Finally a Angular ( v2 ) website is built around the api to display the information using plot.ly for the graphs and angular material for the UI design.
